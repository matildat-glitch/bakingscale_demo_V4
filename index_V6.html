Weight Simulator

Operation Guide (V6):

The Weight Input area simulates real weighing conditions, and each added weight can be selected and removed individually.

  • TARE: In record mode, short press = Add ingredient; long press 1.5s = Tare as bowls. In normal mode, any press zeros the scale.
  • Record limit rule: When icon_tare05 is lit, long press TARE has no effect (all tare icons 01-05 are already used).
  • POWER: short press switch unit / long press power off
  • UP: Browse next / Long press 1.5s = Toggle normal mode
  • DOWN: Browse previous / Long press 1.5s = Delete records
